# 个人博客网站开发计划

## TL;DR

> **快速 summary**: 基于 Vue 3 + Tailwind CSS 4 + TypeScript 开发一个极简风格的个人博客网站，参考 Antfu 博客的设计理念（单栏居中、黑白灰配色、大标题、大量留白）。

> **交付成果**: 
> - 完整的博客网站源码
> - 文章管理（本地 Markdown）
> - 暗色模式支持
> - 响应式布局

> **预计工作量**: Medium
> **并行执行**: YES - 3 waves
> **关键路径**: 项目初始化 → 核心组件 → 页面实现 → 内容集成

---

## 背景

### 原始需求
用户想要一个个人博客网站，风格参考 Antfu 的博客（https://antfu.me/），要求：
- 简约好看实用
- 极简黑白风格
- 单栏居中布局
- 支持暗色模式

### 技术栈
- Vue 3.5.25 + TypeScript
- Vite 8.0.0-beta.13
- Tailwind CSS 4.2.1
- Vue Router 4.6.4
- Pinia 3.0.4
- @VueUse/core 14.2.1

### Antfu 博客风格分析
- 单栏居中布局，内容两侧有适当留白
- 黑白灰为主色调，简洁干净
- 大标题字体（如 "Anthony Fu" 作为主标题）
- 大量留白，信息密度低
- 简洁导航：Blog, Projects, Talks 等
- 分段式内容，每个部分有标题
- 蓝色或彩色链接点缀

## 工作目标

### 核心目标
创建一个极简风格的个人博客网站，具备以下特点：
1. **视觉简洁** - 黑白灰配色，大量留白
2. **阅读友好** - 单栏居中，舒适的阅读宽度
3. **功能完备** - 文章列表、详情、标签、搜索、暗色模式
4. **开发简单** - 本地 Markdown 管理，无需后端

### 具体交付物
- [ ] 项目初始化和配置
- [ ] 路由配置（首页、文章页、标签页、关于页）
- [ ] 布局组件（导航栏、页脚、内容容器）
- [ ] 首页文章列表
- [ ] 文章详情页（Markdown 渲染）
- [ ] 标签列表页
- [ ] 关于页面
- [ ] 暗色模式切换
- [ ] 搜索功能
- [ ] 示例文章（3-5 篇）

### 完成定义
- [ ] `npm run dev` 可以正常启动开发服务器
- [ ] 首页显示文章列表
- [ ] 点击文章可以进入详情页
- [ ] 标签页可以筛选文章
- [ ] 暗色模式可以正常切换
- [ ] 搜索功能可以过滤文章

### 必须有
- 极简黑白风格设计
- 响应式布局（PC + 移动端）
- 暗色模式
- 本地 Markdown 文章管理

### 禁止有
- 复杂的动画效果
- 过度的视觉装饰
- 评论系统（后续考虑）
- CMS 后端
